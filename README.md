# FitClinic

## What is FitClinic?

FitClinic is a fitness and wellness management platform designed to help fitness professionals, clinics, and individuals track, manage, and optimize their fitness journeys.

## Features

### Core Functionality

- **Client Management**: Manage client profiles, fitness goals, and progress tracking
- **Workout Planning**: Create and customize workout plans tailored to individual needs
- **Progress Tracking**: Monitor fitness metrics, measurements, and achievements over time
- **Nutrition Management**: Track dietary intake and nutritional goals
- **Appointment Scheduling**: Schedule and manage fitness consultations and training sessions
- **Analytics & Reporting**: Generate insights and reports on client progress and outcomes

### User Roles

- **Fitness Professionals**: Trainers, physiotherapists, and wellness coaches
- **Clients**: Individuals seeking fitness guidance and tracking
- **Clinic Administrators**: Manage multiple trainers and client relationships

## Getting Started

### Prerequisites

- Node.js (v18 or later)

### Installation

```bash
# Clone the repository
git clone https://github.com/fitanon/fitclinic.git

# Navigate to the project directory
cd fitclinic

# Install dependencies
npm install

# Run the application
npm start
```

The server will start at `http://localhost:3000`. You can set a custom port via the `PORT` environment variable.

For development with auto-restart on file changes:

```bash
npm run dev
```
