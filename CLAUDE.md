## Project Overview

FitClinic is a fitness and wellness management platform for fitness professionals, clinics, and individuals. It is currently in early-stage development — only a static "Coming Soon" landing page exists (`index.html`). No application framework, backend, database, or build system has been set up yet.

## Current State

- **Single static page**: `index.html` contains the landing page with embedded CSS and JavaScript (email signup form, countdown timer)
- **No build system**: No package.json, no bundler, no framework configured
- **No tests**: No test runner or test files exist
- **No backend**: Technology stack is undecided (see README.md for candidate options)
- **No linting/formatting**: No ESLint, Prettier, or EditorConfig configured

## Development

To preview the site locally, open `index.html` in a browser. There are no build or install steps.

## Planned Features (from README roadmap)

Client management, workout planning, progress tracking, nutrition management, appointment scheduling, analytics/reporting, mobile app, and wearable integration. Technology stack (frontend framework, backend, database) is TBD.
